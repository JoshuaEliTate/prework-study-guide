HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • the br element breaks writing or images
  • HTML is used as the foundation of a webpage
  • The h element is used for headers starting as 1 as the largest

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • to link a CSS script to HTML you must add the file to the code
  • CSS is used to format the size of images and paragraphs/lists
  • to have CSS show up on a webpage you must link it with the HTML

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git add -A: stages the changed files into the directory of Git
  • git commit -m"your edits": commit the files to your repository
  • git pull origin main: updates your branch to match the main
  • git push origin "Branch-Name": pushes your changes into the repository
  • git checkout main: puts you back onto the main branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • To simplify and not repeat your code you can use for loops
  • === returns a Boolean result, true if its equal false if not
  • For an array, you must use the title and its number in the list to access an item from it
